# Celadon for iTerm2

Four themes: `celadon` (dark, the default), `celadon-powder` (dark, low
contrast), `celadon-jade` (dark, high contrast), `celadon-sky` (light).

## Install

Download a preset and open it — iTerm2 imports it automatically:

```sh
curl -O https://raw.githubusercontent.com/celadon-theme/celadon-theme/main/ports/iterm2/celadon.itermcolors
open celadon.itermcolors
```

Then activate it: **Settings → Profiles → Colors → Color Presets… →
celadon**.

(Manual alternative: Color Presets… → **Import…**, then pick the downloaded
`.itermcolors` file.)

Using the [oh-my-posh port](../oh-my-posh/)? iTerm2 doesn't bundle Nerd
Font symbols, so pick a [Nerd Font](https://www.nerdfonts.com) build under
**Settings → Profiles → Text** (the screenshot font installs as
**MonaspiceNe Nerd Font**).

These files are generator output — don't edit them; if a color looks wrong,
[open an issue](https://github.com/celadon-theme/celadon-theme/issues).
