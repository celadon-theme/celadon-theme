# Celadon for Windows Terminal

Four themes: `celadon` (dark, the default), `celadon-powder` (dark, low
contrast), `celadon-jade` (dark, high contrast), `celadon-sky` (light).

## Install

Each `<slug>.json` here is one color-scheme object. Open Windows Terminal
settings (`Ctrl+Shift+,`), paste the object into the `"schemes"` array of
`settings.json`, then set it on a profile:

```json
"colorScheme": "celadon"
```

(Or via UI: Settings → Color schemes appears after the JSON is saved.)

These files are generator output — don't edit them; if a color looks wrong,
[open an issue](https://github.com/celadon-theme/celadon-theme/issues).
